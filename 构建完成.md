## 🚀 快速开始

### 安装 Debug 版本

```bash
# 通过 ADB 安装
adb install app-debug.apk

# 或直接在 Android 设备上打开 APK 文件安装
```

### 签名 Release 版本

```bash
# 1. 生成密钥库（首次需要）
keytool -genkey -v -keystore yunwa-release-key.jks \
  -keyalg RSA -keysize 2048 -validity 10000 \
  -alias yunwa

# 2. 签名 APK
apksigner sign --ks yunwa-release-key.jks \
  --ks-key-alias yunwa app-release-unsigned.apk

# 3. 验证签名
apksigner verify app-release-unsigned.apk

# 4. 安装
adb install app-release-unsigned.apk
```
